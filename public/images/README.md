## Image Requirements

- **Format**: JPG or PNG
- **Resolution**: Minimum 1920x1080 for hero images, 1200x800 for action scenes
- **Aspect Ratio**: 16:9 for hero images, flexible for action scenes
- **Style**: Should match the character's universe aesthetic
  - Peter: Classic comic book style
  - Miles: Street art/graffiti style
  - Gwen: Punk rock aesthetic
  - Noir: 1930s black and white film noir
  - Peni: Anime/mech futuristic style
